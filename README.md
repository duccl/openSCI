# openSCI

openSCI is a Python Continuous Integrator with bash script to deal with [react]("https://reactjs.org/") and [angular]("https://angular.io/") deploys to github

## Installation

Use git to clone this repo

```bash
git clone https://github.com/duccl/openSCI
```

## Usage

```bash
python3 thread-orchestror --projectsRoots "/react/mytodo" "/angular/mytodoang" --projectsLabels "react-proj" "angular-proj" --frameworks "react" "angular"
```
